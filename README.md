# SkillMatcher HHAI

FastAPI проект для сопоставления навыков с использованием ИИ.

## Скачивание модели Llama 3 8B

Для работы с нейросетевыми функциями проекта требуется модель Llama 3 8B.

### Быстрый запуск (Windows)
```bash
# Запустите batch файл для автоматической установки зависимостей и скачивания модели
download_llama.bat
```

### Ручная установка
```bash
# Установите зависимости
pip install -r requirements.txt

# Запустите скачивание модели
python download_llama.py
```

Модель будет скачана в папку `neural_networks/` в формате GGUF (q4_k_m quantization).

### Аутентификация на Hugging Face

Для скачивания некоторых моделей может потребоваться аутентификация на Hugging Face.

#### Рекомендуемый способ (с токеном):

1. Создайте аккаунт на [Hugging Face](https://huggingface.co/join)
2. Перейдите в [настройки токенов](https://huggingface.co/settings/tokens)
3. Создайте новый токен с типом "Read"
4. Скопируйте токен и добавьте в переменные окружения:

```bash
# Создайте файл .env в корне проекта
HF_TOKEN=ваш_токен_здесь
```

Или установите переменную окружения:
```bash
# Windows
set HF_TOKEN=ваш_токен_здесь

# Linux/Mac
export HF_TOKEN=ваш_токен_здесь
```

#### Альтернативный способ (с логином и паролем):

```bash
# Создайте файл .env в корне проекта
HF_USERNAME=ваш_логин_здесь
HF_PASSWORD=ваш_пароль_здесь
```

Или установите переменные окружения:
```bash
# Windows
set HF_USERNAME=ваш_логин_здесь
set HF_PASSWORD=ваш_пароль_здесь

# Linux/Mac
export HF_USERNAME=ваш_логин_здесь
export HF_PASSWORD=ваш_пароль_здесь
```

### Системные требования
- Python 3.8+
- Минимум 8 GB свободного места на диске
- Стабильное интернет-соединение (размер модели ~4.6 GB)
